Privacy Policy

Terakhir diperbarui: 29 Juni 2024

pranikah ("kami", "aplikasi") menghormati privasi Anda. Kebijakan ini menjelaskan bagaimana kami menangani informasi saat Anda menggunakan aplikasi pranikah.

1. Informasi yang Kami Kumpulkan

pranikah adalah aplikasi berbasis browser yang menyimpan data secara lokal di perangkat Anda. Kami tidak mengumpulkan, menyimpan, atau mengirim data pribadi Anda ke server manapun.

Data yang tersimpan di perangkat Anda meliputi:

  • Nama mempelai (yang Anda masukkan sendiri)
  • Tanggal pernikahan
  • Data budget dan checklist persiapan
  • Preferensi bahasa dan mata uang

2. Penyimpanan Data

Semua data disimpan menggunakan Local Storage browser di perangkat Anda. Data tidak dikirim ke server kami atau pihak ketiga manapun. Jika Anda menghapus data browser atau menggunakan fitur "Reset Persiapan", data akan hilang secara permanen.

3. Cookies dan Teknologi Pelacakan

Aplikasi pranikah sendiri tidak menggunakan cookies untuk tracking. Namun, halaman website kami menggunakan layanan pihak ketiga berikut:

  • Google AdSense — untuk menampilkan iklan. Google mungkin menggunakan cookies untuk menampilkan iklan yang relevan. Anda dapat mengelola preferensi iklan di Google Ads Settings.

4. Layanan Pihak Ketiga

Website kami mungkin menggunakan layanan berikut:

  • Google AdSense (iklan)
  • GitHub Pages (hosting)

Masing-masing layanan memiliki kebijakan privasi tersendiri yang kami sarankan Anda baca.

5. Keamanan Data

Karena data tersimpan lokal di perangkat Anda, keamanan data bergantung pada keamanan perangkat Anda sendiri. Kami menyarankan untuk menggunakan password/pin pada perangkat Anda.

6. Hak Anda

Anda memiliki kontrol penuh atas data Anda:

  • Anda dapat melihat semua data Anda di dalam aplikasi
  • Anda dapat menghapus semua data melalui fitur "Reset Persiapan"
  • Anda dapat menghapus data browser kapan saja

7. Perubahan Kebijakan

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Perubahan akan dipublikasikan di halaman ini dengan tanggal pembaruan yang baru.

8. Kontak

Jika Anda memiliki pertanyaan tentang kebijakan privasi ini, silakan hubungi kami melalui halaman Ko-fi.
